# GhostWriter

一个模拟"幽灵作家"概念的 VS Code 代码补全插件。通过 AI 预测代码并以自然打字节奏逐字输出，让代码补全看起来像用户在正常敲代码。

## 功能特点

- 🤖 **AI 代码预测** - 调用 OpenAI 兼容 API 预测用户接下来想写的代码
- ⌨️ **自然打字节奏** - 模拟人类不均匀的打字速度，更加自然隐蔽
- 🔢 **空格合并** - 连续空格一次输出，不逐个空格延迟
- 🔇 **静默运行** - 无弹窗、无状态栏显示，保护隐私

## 使用方法

1. 配置 API Key（在设置中输入）
2. 按 `Cmd/Ctrl + Enter` 触发代码预测
3. 继续敲击键盘，AI 预测的代码会逐步"接管"你的输入
4. 按任意键可打断预测

## 配置选项

### API 配置
- **API URL**: OpenAI 兼容格式的 API Base URL
- **API Key**: 你的 API Key
- **模型名称**: 使用的模型（如 gpt-3.5-turbo）
- **提示词**: 系统提示词
- **最大输出 Token**: 每次预测的最大输出长度

### 输出设置
- **单次按键输出字符数**: 每次按键输出的代码字符数量

### 自然节奏设置
- **启用自然打字节奏**: 模拟人类不均匀的打字速度
- **基础打字延迟**: 基础延迟时间（毫秒）
- **延迟随机波动范围**: 延迟的随机变化范围
- **标点符号速度倍率**: 标点符号的延迟倍数
- **字母间额外随机延迟**: 字母间的额外随机延迟
- **空格键延迟倍率**: 空格的延迟倍数
- **换行延迟倍率**: 换行的延迟倍数
- **启用空格合并**: 连续空格一次输出

## 工作原理

GhostWriter 通过拦截 VS Code 的键盘输入事件，在 READY/TYPING 状态下，将用户输入替换为 AI 预测的代码片段。

## 隐私说明

本插件专为需要隐蔽使用代码补全的场景设计，无弹窗、无明显状态栏提示。

## 键盘快捷键

- `Cmd/Ctrl + Enter`: 触发代码预测
- 任意键: 打断当前预测
